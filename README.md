Mindwave
A generative binaural audio PWA for focus, flow, and rest. No ads, no subscription, no network required after first load. All audio is synthesized locally in the browser via the Web Audio API.

What it does
Plays binaural beats — two sine wave tones at slightly different frequencies, one per ear — which the brain perceives as a pulsing rhythm at the difference frequency. Layered over configurable background noise (brown, pink, or white). Everything generates in real time; there are no audio files to download.

Controls
Tone / Noise sliders — mix between the binaural tone and background noise
Noise type — Brown is warmest; White is brightest; Pink sits between them
Carrier frequency — base pitch of the tones (80–300 Hz). Lower = warmer. Try 120 Hz for sleep, 200–240 Hz for focus.
Session timer — 15, 30, 45, 60 minutes, or infinite. Ends with a soft bowl-strike tone.

Requirements
Headphones are required. Binaural beats work by delivering slightly different frequencies to each ear independently. Speakers mix the signal before it reaches you, so the effect is lost.
